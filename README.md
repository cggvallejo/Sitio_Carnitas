# 🌮 Carnitas La Patrona

Bienvenido al repositorio oficial de **Carnitas La Patrona**, una plataforma moderna, rápida y elegante diseñada para ofrecer la mejor experiencia en la compra de carnitas artesanales.

---

## ✨ Características Principales

- **Diseño Premium**: Interfaz fluida con estética de "Liquid Glass" y animaciones refinadas.
- **Carrito de Compras**: Experiencia de usuario intuitiva para gestionar pedidos.
- **Chatbot Inteligente**: Chatbot integrado para asistencia en tiempo real y toma de pedidos.
- **Notificaciones por WhatsApp**: Integración directa para confirmar pedidos y mantener el contacto con el cliente.
- **Responsive Design**: Optimizado para dispositivos móviles, tablets y escritorio.

## 🛠️ Tecnologías Utilizadas

- **Frontend**: [React 19](https://react.dev/), [Vite](https://vitejs.dev/), [Tailwind CSS](https://tailwindcss.com/) (opcional/si se aplica), [Framer Motion](https://www.framer.com/motion/).
- **Backend**: [Node.js](https://nodejs.org/), [Express](https://expressjs.com/).
- **Herramientas de Calidad**: [ESLint](https://eslint.org/).

## 🚀 Instalación y Configuración

Sigue estos pasos para ejecutar el proyecto localmente:

### Requisitos Previos

- [Node.js](https://nodejs.org/) (v18 o superior recomendado)
- [npm](https://www.npmjs.com/)

### Pasos

1. **Clonar el repositorio**:
   ```bash
   git clone https://github.com/tu-usuario/carnitas-el-patron.git
   cd carnitas-el-patron
   ```

2. **Instalar dependencias del Frontend**:
   ```bash
   npm install
   ```

3. **Instalar dependencias del Servidor**:
   ```bash
   cd server
   npm install
   cd ..
   ```

4. **Variables de Entorno**:
   Copia el archivo `.env.example` a `.env` y configura las variables necesarias:
   ```bash
   cp .env.example .env
   ```

## 💻 Uso

### Desarrollo

Inicia el entorno de desarrollo para el frontend y el servidor:

```bash
# Terminal 1: Frontend
npm run dev

# Terminal 2: Servidor
cd server
npm start
```

### Producción

Para generar el build de producción del frontend:

```bash
npm run build
```
